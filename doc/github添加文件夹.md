使用git add dir/files的方式添加文件  


知道2种较简单的在线操作方法...  
方法1：先点击New file按钮，然后输入含有slash字符("/")的文件名即可，比如在文件名的输入框输入，即会在文件名自动产生新文件夹...  

方法2：先点击Upload files按钮，然后直接把本地的文件夹拖进去批量上传，当然上传前要上传的文件需要放进该文件夹...
